Answers you can audit.

A browser-only AI workbench. 01

What makes it different

// every chip below is LIVE today
A1

Zero backend LIVE

The whole product is static files on GitHub Pages. There is no server of ours to receive your data — file contents live in tab memory and vanish on close.

// zero egress to us · metadata-only persistence
A2

BYO keys LIVE

Requests go straight from your browser to your provider — Anthropic, OpenAI, or any OpenAI-compatible endpoint — under your own account. Keys live in localStorage, one per provider.

// anthropic · openai · custom endpoint
A3

Local engine LIVE

The LOCAL provider answers with no key, no AI, and zero network. It runs a real investigation over the project index and returns findings through the same trace UI, verdict-labeled KNOWN LOCALLY.

// def · refs · imports · structure · tests
A4

Single-file LIVE

Dependency-free HTML, CSS and ES modules — no build step, no framework, no npm. The workbench loads zero third-party scripts and its CSP blocks inline script outright.

// app.html · index.html · dna.html
A5

Multi-language indexing LIVE

Symbol & import extraction across JS/TS, Python, Go, Rust, Java, Kotlin, Ruby, C#, Swift, PHP — dynamic import(), tsconfig paths, workspaces, receiver methods, use/mod, Maven/Gradle roots, namespace maps, composer psr-4. Regex-based, no parser, no dependencies.

// 10+ languages today — others basic
A6

Exportable traces LIVE

Export the whole session — answers, traces, and the actual cited lines — as Markdown or a self-contained zero-asset HTML page. Or copy one answer, or one evidence chip's path:line + quote.

// markdown · self-contained html · per-chip copy
02

How it works

// load → ask → trace
01 · LOAD

Your project, in memory

Drag a folder in (or use the File System Access API). MERIDIAN indexes symbols, import edges, entry points, tests and packages, then shows a PROJECT INTELLIGENCE overview. Saved projects persist metadata only — never contents.

02 · ASK

LOCAL engine or your model

Ask in plain language. The LOCAL engine answers structure, definitions and references with no network. Connect a key and the smart context engine sends only the relevant, grounded evidence — never the whole repo.

03 · TRACE

Follow the evidence

Every answer carries a verdict and a trace. Click an evidence chip to open the cited file at the cited line. When you're done, export the session as Markdown or a self-contained HTML page.
