Questions

// asked, answered, traced

Yes — as of this beta, the workbench is a real, working instrument: it loads your project into your browser's memory and answers with live evidence traces. Two honest footnotes: the trace console on this page is still a scripted simulation (it wears a SIM chip), and the capabilities marked ROADMAP don't exist yet. Meridian began as a demonstration of the Lucid Engine design system; the demonstration got promoted.

In your browser. We run no servers, so your files, questions, and API key never reach us — egress to us is zero bytes by architecture, not by policy. When you ask a question, your browser sends it (plus the files the context engine selected) directly to your chosen provider's API — Anthropic by default; OpenAI or a custom OpenAI-compatible endpoint in settings — under your own account. Details in the privacy policy.

A modern browser and your own API key — Anthropic by default, or OpenAI / any OpenAI-compatible endpoint from settings. Meridian itself is free during the beta; your model usage is billed by your provider to you, at their rates. We recommend creating a key with a spend limit. Keys are stored in your browser's localStorage, one per provider, and sent to exactly one place: that provider's API endpoint. No API key at all? MERIDIAN is a deterministic project-intelligence engine first — it indexes your project (symbols, imports, entry points, tests, structure) and the LOCAL provider answers questions about definitions, references, relationships and structure with no key, no AI, and no network. A model is optional, and only for interpretation.

Every answer carries its reasoning path — the steps, the evidence, the confidence. Evidence chips open the cited file at the cited lines, so you can check the model's work. And you should: traces make verification fast, but they're machine-generated too, and can be wrong. If a trace can't be parsed, the answer says so instead of pretending.

The list is how you hear about what ships next — deeper Instruments, Drift Watch, and the planned paid tiers when they become real. It's small on purpose and written by a person, not a sequence.

When the waitlist is live, your address (and tier interest, if you pick one) is sent to Formspree, a form backend, and used for exactly one thing: contacting you about Meridian. No newsletters you didn't ask for, no resale, no ad pixels. A copy is mirrored in your own browser's localStorage so this page remembers you — the [ CLEAR LOCAL ENTRY ] button removes it. Want off the list entirely? Reply to any message and you're gone. Full detail: privacy policy.

The beta is live. The list is for what's next.

The workbench is free to use right now — no signup required. Leave an address to hear when roadmap features and paid tiers become real. You'll hear from a person, not a sequence.
